# Welcome to MolCrafts Project!

## What is MolCrafts?
MolCrafts is a modern project designed to serve as a dedicated platform for computational chemistry researchers. Our mission is to empower scientists by providing them with a streamlined environment to focus on their scientific pursuits without being bogged down by the complexities of tech stacks.

## Family of Molcrafts

### work in progress
- [MolPy](https://github.com/MolCrafts/molpy): Abstract representations for common concepts in computational chemistry
- [MolPot](https://github.com/MolCrafts/molpot): High-performce molecular potential training and deploying framework
- [MolExp](https://github.com/MolCrafts/molexp): Experiment management and workflow orchestration
- [MolVis](https://github.com/MolCrafts/molvis): Interactive visualization tools

### Coming soon
- **MolBuilder**: Build complex model by using LEGO-like way
- **MolPack**: Next generation molecule packing and configuration initializer
- **MolAna**: Simulation analysis and data processing
- **MolBase**: Database for storing and sharing simulation data
